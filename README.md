动画番组及其放送、资讯站点数据集合

## 使用方法

```bash
npm install bangumi-data
```

```js
const bangumiData = require('bangumi-data');
```

也可通过 CDN `https://unpkg.com/bangumi-data@0.3/dist/data.json` 获取 `v0.3.x` 的最新数据
